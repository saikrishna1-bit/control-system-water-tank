# Real-Time Water Tank Control System
## Overview
This project focuses on designing and implementing a PID control system for real-time water level regulation.

## Tools Used
- MATLAB / Simulink
- Arduino Uno R3
- LabVIEW
- Ultrasonic Sensor (HC-SR04)

## Description
A closed-loop control system was developed to maintain water level stability using sensor feedback and PID control.

## Key Features
- Real-time control using feedback loop
- PID tuning for stability
- Hardware and software integration

## Results
- Stable system response
- Reduced overshoot
- Improved accuracy

## Learnings
- Practical control system implementation
- System integration
- Performance evaluation
